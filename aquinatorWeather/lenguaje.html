Python como lenguaje.

Dicha Api combina tres Apis, entre las que se encuentra Twitter y OpenWeatherMap, para dar como resultado el tiempo de tu ciudad.


Realiza una conexión al Twitter de @aquinatoruca, comprueba que tenga un tweet nuevo y coge la ciudad nombrada en dicho tweet. Una vez almacenada la ciudad, se conecta a OpenWeatherMap para obtener el json con todos los datos de dicha ciudad en el día. Finalmente seleccionamos ciertos parametros de dicho json y lo devolvemos a través de un tweet a la persona que escribió el tweet.
