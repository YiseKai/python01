El programa trata de un chat por el que se puede transferir archivos. 

Se necesita lanzar un terminal para chat.py y otro para servidor.py de manera que todos los que esten en el chat actuen también como servidores. 
Para mandar un archivo se debe introducir en el chat el carácter '#', luego introducir por teclado el nombre del archivo (sin comillas dobles) y posteriormente la dirección ip del servidor a la que vas a mandarlo. 

Para que funcione la transferencia del archivo a mandar, este debe llamarse "a.txt". 

Este programa podría mejorarse para poder transferir cualquier archivo que no esté definido en el código.
